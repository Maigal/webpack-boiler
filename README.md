# webpack-boiler

Webpack boilerplate with ES6/SASS/Image copying/Dev server

**Run dev server**

    npm start
    
**Build project**

    npm run build

**File structure**

```bash
**webpack-boiler**
│   package.json
│   webpack.config.js
│
└───src
    │   app.js
    │   app.scss
    │   index.html
    │
    └───images // images folder will be fully copied by default
            icon.png
```
